# 異常系テスト確認結果

## 稼働拡大後の停止条件（厳守）

- 再描画崩れ
  - 目安: 即停止
  - 判定: 一覧・詳細・Dashboard・Audit Log の表示崩れが再操作後も残る
- message 不整合
  - 目安: 条件付き停止
  - 判定: 成功/失敗/認証失敗 message が API 結果と一致しない。単発は監視、再発で停止
- 状態更新不整合
  - 目安: 即停止
  - 判定: 画面間で connectionStatus / freezeStatus が不一致のまま収束しない
- Audit Log 更新失敗の継続発生
  - 目安: 条件付き停止
  - 判定: 単発欠落は継続監視。連続欠落または同種再発で停止
- Settings 保存不整合
  - 目安: 即停止
  - 判定: 保存成功後の値不一致、または保存失敗時の不正な state 更新
- 危険操作 fail-safe 不備
  - 目安: 即停止
  - 判定: confirmed 未チェック/認証不足で危険操作が通過する
- LockGuard 誤作動
  - 目安: 条件付き停止
  - 判定: 単発誤作動は切り戻し+監視。再発または中断不能化で停止

### 判定ラベル運用

- 即停止: 拡大運用を停止し、解放済み範囲をこれ以上広げない
- 条件付き停止: 再発条件を監視し、満たした時点で停止へ移行
- 継続監視: 単発事象として記録して監視サイクルで追跡

#### Level1 参照系 停止/巻き戻し判定（詳細）

- 一覧が表示されない: 即停止
  - 判定目安: 初期表示または再取得後に一覧領域が空白のまま復帰しない
- 詳細が表示されない: 即停止
  - 判定目安: 一覧選択しても詳細パネルが更新されない状態が継続する

  - 判定目安: 一覧件数・状態内訳とDashboard数値が継続的に一致しない

  - 判定目安: ログ行の欠落・列崩れ・順序崩れで監視利用ができない
- Settings 参照が崩れる: 条件付き停止
  - 判定目安: 単発の参照欠落は継続監視、再発時は停止

- プレースホルダー復帰しない: 条件付き停止
  - 判定目安: 失敗時に復帰しない事象が再発した場合は停止、単発は継続監視

  - [ ] confirmed ステップを踏まずに status が戻る事象がない
  - [ ] status が元の値に戻り3画面で一致している
  - [ ] Audit Log に action / target / result が欠落なく記録されている

### Export

- 見直し候補に残している理由: 認証不足通過・LockGuard 残留のいずれかが発生した場合は即時全面見直し相当であり、継続監視が外せない
- 再評価条件: 5回連続で認証チェック完全・LockGuard 正常閉鎖・Audit Log 整合がすべて確認でき、報告ゼロが続いた場合
- 再評価前に再確認する項目
  - [ ] 認証不足のユーザーで Export が実行されない
  - [ ] LockGuard が閉じている／処理後残留がない
  - [ ] Audit Log に Export 実行結果が正しく記録されている

### Delete

- 見直し候補に残している理由: Export の再評価条件を満たした後でなければ Delete 単独での再評価を開始できない
- 再評価条件: Export の再評価条件を満たした上で、Delete 固有の削除後3画面整合が5回確認できた場合
- 再評価前に再確認する項目
  - [ ] Export の再確認項目がすべて通っている
  - [ ] 削除後に一覧・詳細・Dashboard の3画面で対象が消えている
  - [ ] Audit Log に Delete 実行結果が欠落なく記録されている

### 巻き込み確認（一覧/詳細/監査ログ/再描画整合）

- 見直し候補に残している理由: 条件付き運用機能の実行後に参照系・監査表示へ波及異常が発生することがある
- 再評価条件: 上記5機能の再評価条件をすべて満たした回で参照系の整合も同時確認
- 再評価前に再確認する項目
  - [ ] 一覧・詳細が崩れず操作結果を正しく反映している
  - [ ] Audit Log が欠落なく読める
  - [ ] 再描画後に不整合が残らない
