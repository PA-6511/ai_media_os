## Level4 継続判定フェーズ 継続・条件付き継続・停止候補（固定）

> Level4（Export / Delete）は最終危険操作段階。少しでも危険な兆候があれば即停止寄りで判定する。

### 継続してよい条件

- Export
  - confirmed・password・twoFactorCode・confirmationText が全て通過した場合のみ実行される
  - 実行後に Audit Log が欠落なく追記される
  - 再描画・状態整合に崩れが出ない
- Delete
  - 上記と同条件を満たす
  - 削除後に一覧・詳細・管理パネルが同時に整合する

### 条件付き継続の条件

- message 表示に軽微な不整合が単発で出たが、再発していない
  - 追加監視条件: 次回も同箇所を重点確認、再発で停止に格上げ
- 再描画に軽微な崩れが単発で出たが、消滅・空白化には至っていない
  - 追加監視条件: 次回も同箇所を重点確認、再発または範囲拡大で停止に格上げ

### 停止候補の条件

- confirmed 未チェックで Export / Delete が進んだ
  - 理由: fail-safe の根幹が破綻している
- password / twoFactorCode / confirmationText のいずれか不足で実行が通った
  - 理由: 認証チェックが機能していない
- Audit Log が増えない
  - 理由: 実行証跡が残らず運用継続不可
- 一覧・詳細・管理パネルの3画面で状態が一致しない
  - 理由: データ整合性が失われている
- LockGuard が閉じない・誤作動した
  - 理由: 安全装置が正常動作していない
- 巻き込み確認対象（下記）のいずれかに崩れが出た
  - 一覧表示 / 詳細表示 / Dashboard / Audit Log 参照 / Settings 参照 / Settings 保存 / Connect / Disconnect / Freeze / Unfreeze
  - 理由: Level1〜3 解放機能まで影響が及んでおり全体停止が必要
