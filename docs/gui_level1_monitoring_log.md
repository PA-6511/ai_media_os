## Level1 継続判定（初回）

### 継続してよい

- 一覧表示: 継続表示と再読込後の表示維持が確認しやすい
- 詳細表示: 一覧選択との連動確認が可能で副作用が小さい
- Dashboard: 参照系集計の整合監視を継続しやすい
- Audit Log 参照: 監視基盤として継続利用が必要

### 条件付き継続

- Settings 参照: 参照値の空欄化や初期値固定の再発有無を条件に継続

### 停止候補

- 一覧表示: 一覧消失や継続的な行欠落が出た場合は停止候補
- 詳細表示: 詳細表示不能や選択不整合の再発時は停止候補
- Dashboard: 数値空欄化や一覧との不整合継続時は停止候補
- Audit Log 参照: 一覧崩れや読取不能状態が継続する場合は停止候補
- Settings 参照: 参照値空欄化が連続発生する場合は停止候補

### Level1 判定対象外（別Levelで判定）

- 更新系: Settings 保存 / Connect / Disconnect
- 条件付き更新系: Freeze / Unfreeze
- 危険操作系: Export / Delete
