## Level1 継続観測項目（固定）

- 一覧表示の安定性
	- 何を見れば正常か: 画面遷移・再読込後も一覧の行/列構造が崩れない
- 詳細表示の安定性
	- 何を見れば正常か: 一覧選択に連動して詳細が遅延なく更新される
- Dashboard 表示の安定性
	- 何を見れば正常か: 一覧件数・状態内訳と Dashboard 表示が一致する
- Audit Log 一覧表示の安定性
	- 何を見れば正常か: ログ一覧に欠落や順序崩れがなく表示される
- Settings 参照表示の安定性
	- 何を見れば正常か: 設定値が空欄化せず取得値どおりに表示される
- message 表示の有無
	- 何を見れば正常か: 参照系の成功/失敗に応じた message が表示される
- プレースホルダー復帰の有無
	- 何を見れば正常か: 詳細取得失敗時にプレースホルダーへ戻り再取得で復帰する

## Level1 条件付き継続 追加監視条件

### 一覧表示

- 継続条件: 一覧表示が継続し、再読込後も表示欠落がない
- 追加で見るべき指標: 行欠落件数、表示空白化の発生回数、再表示所要時間
- 監視頻度: 監視サイクルごとに3回確認
- 継続中止条件: 一覧消失または行欠落が連続発生する

### 詳細表示

- 継続条件: 一覧選択と詳細表示の対象が一致し続ける
- 追加で見るべき指標: 選択不整合件数、詳細更新遅延、詳細表示失敗回数
- 監視頻度: 監視サイクルごとに3回確認
- 継続中止条件: 詳細未表示または選択不整合が再発する

### Dashboard

- 継続条件: 一覧件数・状態内訳とDashboard表示が一致する
- 追加で見るべき指標: 数値乖離件数、空欄化回数、更新遅延
- 監視頻度: 監視サイクルごとに3回確認
- 継続中止条件: 数値空欄化または不整合が継続する

### Audit Log 参照

- 継続条件: 一覧表示が崩れず、内容を継続して読める
- 追加で見るべき指標: 行/列崩れ件数、表示欠落件数、順序不整合件数
- 監視頻度: 主要確認操作ごとに毎回確認
- 継続中止条件: 読取不能な崩れ、欠落、順序崩れが再発する

### Settings 参照

- 継続条件: 参照値が空欄化せず安定表示される
- 追加で見るべき指標: 空欄化回数、初期値固定回数、再取得後一致率
- 監視頻度: 監視サイクルごとに3回確認
- 継続中止条件: 空欄化または初期値固定が連続発生する
